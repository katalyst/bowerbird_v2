Bowerbird 2
===========

Bowerbird is a wrapper for Bootstrap (via [Bootstrap for Sass](http://github.com/thomas-mcdonald/bootstrap-sass)).

Using Bowerbird
---------------

Add to `Gemfile`:

```ruby
gem "bowerbird_v2", :git => "git@github.com:katalyst/bowerbird_v2.git"
```

Add to `application.scss`:

```sass
@import "bowerbird";
```
